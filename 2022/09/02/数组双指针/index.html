    两数之和

    排序

    • 使数组成为有序数组

    左右指针

    • 左右指针分别指向数组第一个元素和数组最后一个元素
      • num[left]+num[right]>target:right--;
      • num[left]+num[right]<target:left++;
    public int[] twoSum(int[] numbers, int target) {
    int left = 0;
    int right = numbers.length - 1;
    while (left < right) {
      int p = numbers[left] + numbers[right];
      if (p == target) {
        return new int[]{left+1, right+1};
      } else if (p > target) {
        right--;
      } else {
        left++;
      }
    }
    return null;
}

    三数之和

    two sum 改造

    • 增加一个搜索范围, lo, hi
    • 成功找到一组后, lo 从下一个不同的地方开始
    public List<List<Integer>> twoSum(int[] nums, int lo, int hi, int target) {
    List<List<Integer>> result = new ArrayList<>();
    int left = lo;
    int right = hi;
    while (left < right) {
        int nleft = nums[left];
        int nright = nums[right];
        int p = nleft + nright;
        if (p == target) {
            List<Integer> tmp = new ArrayList<>(Arrays.asList(nleft, nright, nums[lo-1]));
            result.add(tmp);
            while (nums[left] == nleft && left < right) {
              left++;
            }
        } else if (p > target) {
            right--;
        } else {
            left++;
        }
    }
    return result;
}

    总体思路

    • 对数组排序

    • 外层循环遍历数组, i=[0, length-3]

    • twoSum(numbers, i+1, length-1, target-nums[i])

      • 如果返回值为 null, 外层继续遍历
      • 如果返回值不为 null, 加入结果集, 继续遍历

public List<List<Integer>> threeSum(int[] nums) {
    List<List<Integer>> result = new ArrayList<>();
    Arrays.sort(nums);
    for (int i = 0; i < nums.length-2; i++) {
        int num = nums[i];
        List<List<Integer>> twoSumResult = twoSum(nums, i+1, nums.length-1, 0-num);
        if (!twoSumResult.isEmpty()) {
            result.addAll(twoSumResult);
            while (i+1 < nums.length-2 && num == nums[i+1]) {
              i++;
            }
        }
    }
    return result;
}

    原地操作数组

    移除元素

    • 双指针, lo 指向待写入的下标, hi 指向写入值对应的下标
    public int removeElement(int[] nums, int val) {
    int lo = 0;
    int hi = 0;
    while (hi < nums.length) {
      if (nums[hi] != val) {
        nums[lo] = nums[hi];
        lo++;
      }
      hi++;
    }
    return lo;
}

    移动零

    • 移除值为 0 的元素, 然后将 [lo,nums.length-1] 都置为 0
    public void moveZeroes(int[] nums) {
    int lo = 0;
    int hi = 0;
    while (hi < nums.length) {
      if (nums[hi] != 0) {
        nums[lo] = nums[hi];
        lo++;
      }
      hi++;
    }
    for (int i = lo; i < nums.length; i++) {
      nums[i] = 0;
    }
}

    删除数组当中的重复项

    • 数组是否为有序?
      • 无序先排序
      • 有序, lo 指向待写入的下标(待写入的下标为被比较值下标+1), hi 指向待写入的值
    public int removeDuplicates(int[] nums) {
    int lo = 0;
    int hi = lo + 1;
    while (hi < nums.length) {
      // nums[lo]->被比较值
      if (nums[hi] != nums[lo]) {
        // 指向待写入的下标
        lo++;
        nums[lo] = nums[hi];
      }
      hi++;
    }
    return lo+1;
}

    最长回文子串

    回文子串的定义

    • 回文子串有两种, 奇数长度 or 偶数长度
      • 奇数长度, 双指针从同一个中点像外扩散, 直到两个指针指向的值不相等
      • 偶数长度, 双指针从相邻的中点像外扩散, 直到两个指针指向的值不相等
    • 由于最长回文子串的中点有可能是原始字符串中的任意一个元素, 所以外层需要遍历 s 的每一个字符

    思路: 遍历原始字符串中每个元素, 尝试以当前元素(奇偶情况)作为回文子串的中点并向外扩散

    
private String longest(String s, int i, int j) {
    int len = s.length();
    while (i >= 0 && j < len && s.charAt(i) == s.charAt(j)) {
        i--;
        j++;
    }
    return s.substring(i+1, j);
}

public String longestPalindrome(String s) {
    String max = "";
    for (int i = 0; i < s.length(); i++) {
        String result1 = longest(s, i, i);
        if (max.length() < result1.length()) {
            max = result1;
        }
        if (i+1 < s.length()) {
            String result2 = longest(s, i, i+1);
            if (max.length() < result2.length()) {
                max = result2;
            }
        }
    }
    return max;
}
