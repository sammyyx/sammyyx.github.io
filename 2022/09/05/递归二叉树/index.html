    前/中/后序遍历

    递归

    • 只要是递归调用, 就存在前序和后序
      • 前序指递归前
      • 后序指递归后
    • 由于二叉树需要先后递归左右子树, 在两次递归之间称为中序
    private List<Integer> track = new ArrayList<>();

private void traverse(TreeNode root) {
    if (root == null) {
        return;
    }
    track.add(root.val);
    traverse(root.left);
      // 中序遍历 track.add(root.val);
    traverse(root.right);
      // 后序遍历 track.add(root.val);
}

public List<Integer> preorderTraversal(TreeNode root) {
    traverse(root);
    return track;
}

    二叉树的深度

    最大深度: 根节点到「最远」叶子节点的最长路径上的节点个数

    思路: 遍历所有「叶子节点」, 比较「根节点」到所有「叶子节点」的长度, 取最大的一个

    private int maxDepth = 0;

private void traverse(TreeNode root, int depth) {
    if (root == null) {
        return;
    }
    if (root.left == null && root.right == null) {
        maxDepth = Math.max(maxDepth, depth);
    }
    traverse(root.left, depth+1);
    traverse(root.right, depth+1);
}

public int maxDepth(TreeNode root) {
    traverse(root, 1);
    return maxDepth;
}

    利用后序位置搜集子树的信息

    统计二叉树节点个数

    由于后序是在递归之后发生, 可以理解后序是站在根节点观察左右子树, 在这个位置可以拿到左右子树的遍历结果.

    当前根节点对应的子树包含的节点个数 = 「左子树」的节点个数 + 「右子树」的节点个数 + 1「根节点」

    private int count = 0;

private int traverse(TreeNode root) {
    if (root == null) {
        return 0;
    }
    int left = traverse(root.left);
    int right = traverse(root.right);
    // 「左子树」的节点个数 + 「右子树」的节点个数 + 1「根节点」
    count = Math.max(left+right+1, count);
    return left+right+1;
}

public int countNodeOfBinaryTree(TreeNode root) {
    traverse(root);
    return count;
}

    二叉树的最长直径

    「当前子树的最长直径」= 其「左子树」的深度 + 其「右子树」的深度

    
private int diameter = 0;

private int traverse(TreeNode root) {
    if (root == null) {
          return 0;
    }
    int left = traverse(root.left);
    int right = traverse(root.right);
      // 「左子树」的深度 + 其「右子树」的深度
    diameter = Math.max(left+right, diameter);
    return Math.max(left, right) + 1;
}

public int diameterOfBinaryTree(TreeNode root) {
    traverse(root);
    return diameter;
}
